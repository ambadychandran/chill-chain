#### Install Packages

```
composer install
```

#### Set Database Detail on .env file

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db
DB_USERNAME=root
DB_PASSWORD=
```

#### Set migration with below command

```
php artisan migrate
```

#### Install NPM Dependencies

```
npm install
```

#### Start Laravel Project

```
php artisan serve
```

#### Start Vue Frontend App

```
npm run watch
```

#### Open the URL in the browser

```
http://127.0.0.1:8000
```
